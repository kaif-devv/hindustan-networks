# Hindustan Networks LLP

> Comprehensive Network & Communication Solutions

Official website for **Hindustan Networks LLP** — a leading provider of end-to-end network and communication infrastructure solutions serving businesses in and around Shadnagar, Telangana since 2017.

🌐 **Live site:** https://kaif-devv.github.io/hindustan-networks/

---

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** (build tool)
- **TailwindCSS v4**
- **Framer Motion** (animations)
- **Lucide React** (icons)
- **React Hook Form** + **Zod** (contact form validation)
- **React Helmet Async** (SEO)
- **Radix UI** primitives

## Sections

- Hero — animated particle background
- About — company info and stats
- Services — 15 network service cards
- Industries — 6 industry segments
- Strengths — 5 key differentiators
- Clients — logo wall grid
- Vision & Mission
- Contact — form with validation
- Footer

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:5173/hindustan-networks/

## Build & Preview

```bash
npm run build
npm run preview
```
